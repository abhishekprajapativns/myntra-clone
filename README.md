# Myntra Clone 🛍️

This is my first full-stack project where I built both frontend 
and backend from scratch. I took inspiration from Myntra's website 
to build this e-commerce application.


## ✨ What I Built

- Navigation bar with search
- Auto-sliding banner
- Products section with Add to Bag
- Wishlist functionality  
- User Login & Signup with JWT Authentication
- Multi-step Checkout (Address + Payment)
- Responsive Footer

## 🛠️ Technologies Used

**Frontend:**
- HTML5
- CSS3
- Vanilla JavaScript

**Backend:**
- Node.js
- Express.js
- MongoDB & Mongoose
- JWT (JSON Web Token)
- Bcrypt.js (Password Encryption)

## ⚙️ How to Run Locally

**Step 1 - Clone the repo:**
```bash
git clone https://github.com/abhishekprajapativns/myntra-clone.git
```

**Step 2 - Run Backend:**
```bash
cd backend
npm install
node server.js
```

**Step 3 - Open Frontend:**
- Open `frontend/index.html` in your browser

## 📚 What I Learned

- How to build a REST API using Node.js and Express
- How to connect MongoDB database using Mongoose
- How to implement JWT based authentication system
- How to connect frontend with backend using Fetch API
- How to store and manage data using localStorage
